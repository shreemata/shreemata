⚙️ Commission Settings

Configure commission percentages for the dual-system referral and tree placement model

💡 Dual System:
Direct Commission: Based on referral relationships (who referred whom)
Tree Commission: Based on tree placement (who is under whom in tree)
Total: Always 10% of every order (Direct + Tree + Trust + Development)

💰 Direct Commission

Commission paid to direct referrer (based on referral code)

🏦 Trust Fund

Base allocation + commissions when no referrer + suspended users

🚀 Development Fund

Base allocation + remainder from tree commission pool

🌳 Tree Commission Pool

Distributed to tree parents based on tree placement

💸 Minimum Withdrawal

Minimum amount users can withdraw

🌳 Tree Placement Threshold

Minimum purchase amount required for tree placement (0 = no minimum)
Note: Users who purchase below this amount will receive commissions but won't be placed in the tree visual until they reach the threshold.

📊 Tree Commission Levels

Configure percentage for each tree level (based on tree placement, not referral chain)

🌳 Tree Placement System:
• Tree commissions flow to tree parents (treeParent relationships)
• Level 1: Direct tree parent gets highest percentage
• Level 2: Tree grandparent gets next percentage
• Virtual cards redirect commissions to original users
• Unused remainder goes to Development Fund
